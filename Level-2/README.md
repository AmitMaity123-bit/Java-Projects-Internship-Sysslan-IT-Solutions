# Level 2 - Java Internship Projects

## Overview

This repository contains **Level 2 Java projects** developed as part of the internship program at **Sysslan IT Solutions**. These projects demonstrate intermediate-level Java programming skills including file handling, object-oriented logic, algorithms, and console-based applications.

---

## Projects Included

Level 2 consists of the following tasks:

### 1. Tic Tac Toe Game

A two-player console-based game that allows users to play Tic Tac Toe with automatic win/draw detection.

**Key Features:**

* Player vs Player gameplay
* Board display system
* Win and draw detection
* Input validation

---

### 2. Password Strength Analyzer

A program that evaluates the strength of a password based on security rules.

**Key Features:**

* Checks uppercase, lowercase, digits, and special characters
* Strength scoring system
* Outputs WEAK, MEDIUM, or STRONG

---

### 3. File Encrypt & Decrypt System

A simple file security tool using character shifting (Caesar Cipher technique).

**Key Features:**

* File reading and writing
* Encryption using character shift
* Decryption of encrypted data
* File output generation

---

### 4. Personal Expense Tracker

A console-based application to manage and track daily expenses.

**Key Features:**

* Add and store expenses in a file
* View saved expenses
* Calculate total spending
* Persistent data storage

## How to Run Any Project

### Step 1: Navigate to the project folder

```bash
cd Task-1-TicTacToe
```

### Step 2: Compile the Java file

```bash
javac FileName.java
```

### Step 3: Run the program

```bash
java FileName
```

(Replace `FileName` with the actual class name of each project)
