# 📌 Projects Included

# 1. Number Grid Search Application

## Description

A 3×3 matrix-based application where users can input numbers, search for specific values, and view their positions within the grid.

## Features

* 3×3 matrix creation
* Number search functionality
* Displays row and column positions
* Counts occurrences
* Visual match highlighting
* Input validation

## Key Concept

* 2D Arrays
* Matrix traversal
* Searching algorithms

---

# 2. Temperature Converter

## Description

A menu-driven program that converts temperature between Celsius and Fahrenheit with real-time weather interpretation.

## Features

* Celsius ↔ Fahrenheit conversion
* Continuous menu-based interaction
* Input validation
* Weather condition suggestions

## Formulas

F=(C\times\frac{9}{5})+32

C=(F-32)\times\frac{5}{9}

## Key Concept

* Functions
* Mathematical operations
* Conditional logic

---

# 3. Student Grade Average Calculator

## Description

A system that calculates student performance based on multiple subjects, generates grades, and identifies top and lowest performers.

## Features

* Multiple students & subjects support
* Average calculation
* Grade assignment (A+ to F)
* Pass/Fail system
* Top performer detection
* Performance report generation

## Grade System

* A+ → 90–100
* A → 80–89
* B → 70–79
* C → 60–69
* D → 50–59
* E → 40–49
* F → Below 40

## Key Concept

* Arrays
* Data processing
* Report generation

---

# 4. Secure Password Generator

## Description

A secure password generator that creates strong passwords based on user-selected criteria and evaluates password strength.

## Features

* Custom password length (8–64)
* Include/exclude:

  * Lowercase
  * Uppercase
  * Numbers
  * Special characters
* Remove ambiguous characters
* Generates 5 passwords at once
* Password strength evaluation

## Character Sets

* a–z
* A–Z
* 0–9
* Special symbols

## Strength Levels

* Weak
* Fair
* Good
* Strong
* Very Strong

## Key Concept

* Randomization
* Security basics
* String manipulation
